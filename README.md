# reactjs-web

This is a starter template for web [reactjs](https://reactjs.org) projects.

## How to use this template

Required:

- nodejs

```bash
$ cd react-web/
$ npm install
$ npm start
```

start server [http://localhost:3000](http://localhost:3000)
